La barre des menus

Les fonctions présentes dans ces menus reprennent souvent ce que les boutons proposent déjà. Je ne vais donc pas m'y attarder. Cependant, il y a là quelques fonctions essentielles à connaître:

Menu File

On y trouve les fonctions classiques de création d'un nouveau projet, d'ouverture d'un projet existant, de sauvegarde, d'importation de fichiers 3DStudio *.3ds. On y trouve également la fonction Merge qui permet de fusionner deux projets gmax; utile pour gérer une bibliothèque d'objets standard tels que des essieux, pantographes...etc.

Une autre fonction très intéressante est Summary Info. Elle liste tous les objets du projet est permet de connaître le nombre de polygones utilisés.

Menu Group

Les fonctions de groupages permettent d'attacher temporairement des objets entre eux pour qu'ils se déplacent comme un seul.

Menu View

La fonction Viewport background permet d'afficher une image en fond d'écran. On peut donc afficher une photo, ou mieux, un plan de l'objet que l'on modélise.

Menu Customize

Ce menu propose différents moyens pour paramétrer l'interface à son goût. Pour faire vite, on laissera ces fonctions de côté et travailler avec la configuration par défaut. Le menu Customize propose aussi les boites de dialogue Unit setup et Preferences qu'il sera nécessaire de configurer pour ne pas avoir de mauvaise surprise dans MSTS.

Menu Train Simulator

Le menu le plus important puisqu'il contient le LOD Manager sans lequel gmax ne pourrait pas exporter de fichiers vers MSTS
