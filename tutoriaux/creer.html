Panneau Create

Ce panneau sert à créer toutes sortes d'objets: Primitives 3D, figures 2D, lumières, caméras, systèmes articulés...

Dans la création de modèles pour MSTS, seuls les deux premières familles d'objets seront utiles.

Mais gmax permet aussi de créer des objets composés très intéressants: Les corps booléens réalisés à partir d'opérations booléennes sur deux objets, les objets extrudés (Loft) ou bien les ShapeMerge.

En revanche les possibilités de créer d'objets paramétriques tels que les courbes de Bézier ou les Nurbs ont été supprimées de gmax par rapport à son grand frère 3DStudio Max.

Options intéressantes offertes par les boutons du panneau Create :

Bouton Geometry
Bouton Shapes
Bouton Helpers
